Group 分组

使用指南

在 app.json 或 index.json 中引入group组件

{
  "usingComponents": {
    "fs-group": "path/components/group/index"
  }
}

  • 基础用法
<fs-group title="基础用法">
  <fs-button>default</fs-button>
</fs-group>

    <p><fs-icon name="home"></fs-icon>自定义标题</p>
  </view>

    <fs-button>default</fs-button>
  </view>
</fs-group>
