Card 卡片

使用指南

  • 确保 components 下引入 behavors/router.js, icon、card组件
  • 在 app.json 或 index.json 中引入组件

{
  "usingComponents": {
    "fs-card": "path/components/card/index",
  }
}

属性

参数 说明 类型 默认值
custom-class 外部样式 String -
title 主标题 String -
sub-title 小标题 String -
img 图片地址(可选) String -
circle 图片圆形 Boolean -
append 附加信息 String -
to 内部跳转链接 String -

插槽

插槽名 说明
append 自定义append内容

事件

event 说明 返回值
click 点击 event
